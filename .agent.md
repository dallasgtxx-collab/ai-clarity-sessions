# AI Clarity Sessions Agent

## Purpose
Use this agent for completing and maintaining the AI Clarity Sessions Next.js project. Focus on shipping a working site end to end, especially theStripe checkout experience, marketing pages, and supporting routes.

## Core responsibilities
- Finish incomplete app features from prior conversation context instead of leaving partial implementations behind.
- Keep the marketing site, checkout API, webhook handling, success/cancel pages, and privacy/refund content consistent.
- Fix root causes rather than patching symptoms.
- Preserve accessible, polished UI patterns and keep the project buildable.

## Preferred workflow
1. Review the existing app structure and confirm what is already implemented.
2. Trace the relevant user flow end to end before making changes.
3. Fix the underlying issue, not just the visible symptom.
4. Verify the result with the project build and relevant runtime checks.

## Project-specific guidance
- This is a Next.js app using App Router.
- Stripe checkout should flow through the server-side API route and use the shared product config.
- Keep product metadata aligned between the landing page, checkout route, webhook handler, and fulfillment logic.
- Prefer using existing environment variables and shared helpers instead of duplicating config.
- Keep the site copy friendly, clear, and beginner-focused.

## Verification standard
Before considering work complete, verify with:
- `npm run build`

If a change affects checkout, also confirm the API route and fulfillment flow remain consistent.

## Example prompts
- "Finish the checkout flow and make sure the app builds."
- "Fix the Stripe integration so the landing page can open checkout successfully."
- "Add or correct a missing project feature from the earlier conversation context."
